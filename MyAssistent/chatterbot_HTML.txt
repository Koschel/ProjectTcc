Conceito HTML: é abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo. Um documento é visto como um conjunto de eventos concorrentes dependentes de tempo (como áudio, vídeo, entre outros), conectados por hiperligações. O padrão é independente de outros padrões de processamento de texto em geral. Deseja continuar?

A primeira linha do HTML se chama DOCTYPE. O Doctype avisa aos browsers, robôs de busca, leitores de tela e outras coisas que tipo de documento e aquele que eles estao prestes a carregar. Existem outros códigos que podemos carregar, por exemplo XML. Por isso o Doctype avisa o browser para que ele saiba como se comportar ao ler o código. Depois começamos com a Tag HTML. Isso quer dizer que todo o que estiver entre as tags é escrito em HTML. Ao lado da palavra HTML tem um atributo (explico o que são atributos mais pra frente) chamado lang, onde indicamos qual o idioma do texto que escreveremos. Na tag Head nós indicamos o título do documento e indicamos a tabela de caractéres que o browser deve usar para renderizar seu texto. Também não se preocupe com isso agora. A tag <title> é muito importante. É com ela que você indica o título do documento. O Google e outros sistemas de busca utilizam essa tag para indicar em suas buscas o título da págin. Isso é muito importante para que você apareça bem nas buscas.  Logo depois da tag de fechamento começamos a tag . Dentro deste elemento é que vamos escrever todo o código HTML do resto do site.
O que é uma tag?
Tags são estruturas de linguagem de marcação contendo instruções, tendo uma marca de início e outra de fim para que o navegador possa renderizar uma página. Há uma tendência nos dias atuais para se usar as tags apenas como delimitadores de estilo e/ou conteúdo, tanto em HTML quanto em XML.
topico 2
O JavaScript é uma linguagem de programação utilizada para o desenvolvimento web. A maioria dos projetos que utilizam essa linguagem de programação também fazem uso do HTML e do CSS, de uma maneira podemos dizer que essas três linguagens tornam-se uma forma de espectro para o desenvolvimento web, sendo que as mesmas são utilizadas com a máxima eficiência. Já possuindo um conhecimento sobre linguagem de programação, ajuda a ter a cognição que o JavaScript é uma linguagem que possui um alto nível, dinâmica, interpretada e não tipada, fortemente propício para um estilo de programação funcional e orientado a objetos Mas para começar a entender o JavaScript, primeiramente devemos conhecer os conjuntos de regras e que designar a forma de escrever e compor o programa. A linguagem do JavaScript, os programas são desenvolvidos um utilizando um superconjunto de caracteres ASCII e ISO-Latin-1, que possui um suporte para a maioria dos idiomas mais utilizados. JavaScript se comparado com as outras tecnologias para desenvolvimento web apresentadas neste projeto possui uma interpretação entre letras maiúsculas e  minúsculas, já os espaços em branco o JavaScript acaba ignorando. Esses exemplos apresentados anteriormente, pode ser interpretada com algumas das formas que o JavaScript trabalha, mas devemos ter o conhecimento das outras regras que existem e as que compõe a sintaxe dessa linguagem de programação.
